python /n,/t,/n/t difference

\t，\n，\n\t，what is the difference?
\t ：empty space for 4 characters in front，same like one Tab keyboard key
\n ：change to next line ,like one Enter keyboard key
\n\t : change to next line and empty space for 4 characters in front
